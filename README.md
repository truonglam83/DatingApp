- **Homepage**: We may view other users' personal information. by buttons. Tonder users can like dislike other profiles by buttons.
    + Block: if we don't like this person. The list will never show this user again
    + Like: If we like another user, then click like. When 2 users like each other, there will be a notification screen and can message each other.

- When two users like each other, the system will notify both of them that they have matched each other.

- **Map**: The map can show the relative location of nearby users within a maximum radius of 3km. We can choose people near us to match

 <br>
 <br>
 - **Chat**: Two users after matching each other have the right to message each other. This message is realtime, can send images, icons, ....
 
 ![image](https://github.com/truonglam83/DatingApp/assets/78529041/d471c80c-10e3-4eeb-8650-2e019d5d3359)

Users can upload images, set avatars, other personal information such as date of birth, interests, ....
